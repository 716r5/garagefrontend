# Garage Website Frontend

This is the repository for the frontend code of Garage@EEE's [website](https://garage-eee.com/), built using React.

## Running The Site Locally

To get started running this locally, follow the steps below:

- Clone the repo
- Install all dependencies needed using `npm install`
- Run the development building with `npm run start`
- Website should be running on `localhost:3000`

## Security

This project implements various security measures to protect user data and prevent common vulnerabilities. Please refer to [SECURITY.md](./SECURITY.md) for detailed security guidelines, best practices, and reporting procedures.

### Quick Security Checklist
- Run `npm audit` regularly to check for vulnerabilities
- Never commit `.env` files or hardcode API keys
- Validate all user inputs on both client and server side
- Review the security guidelines before contributing
